autoenqueue – This attribute should be set at the template level to true or false in order to override the ROES Server’s setting in the Options pane (checkbox for Auto-Enqueue Order Items). In most cases the checkbox in the Server is not selected so autoenqueue is set to true on templates that should bypass this and proceed straight to there queue/the default queue.

outputrender – The outputrender attribute can be set to false on a node inside a template to keep it from being rendered, such as a guide image, frame, or canvas overlay (never needed to be set to true).

outputrenderlayoutimage – The outputrenderlayoutimage attribute can be set to false on a template to instruct the ROES Server to ignore the presence of a layoutimage when rendering the product (never needs to be set to true).

preferredqueue – The preferredqueue attribute is a case sensitive value defined on a template to define what queue to feed into in the ROES Server. It should match the Queue name value in the ROES Server exactly.

printerchannel – The printerchannel attribute can be added on a template to set the Channel entry for printer such as Noritsu MP1600.

printersurface – The printersurface attribute can be added on a template to set the Surface entry for printer such as Noritsu MP1600.

qsssurface – The qsssurface attribute can be added on a template to set the Surface entry for Noritsu Net Order.

sa_pagespan – The sa_pagespan attribute is to define a panoramic template for rendering. Set to 2 and only add if a pano product.

ROES Server for DP2:

isdp2book – The isdp2book attribute

jobtext – The jobtext attribute can be added onto any template, node, or option. The text is passed as-is to Kodak’s DP2 software to affect the job at print time.

retouch – The retouch attribute can be used on an Image Option in ROES to mark the image for retouch in the KPARS module in DP2.

servercommand – The servercommand attribute is used to pass additional operations into DP2 (defined further in the ROES DP2 Server User’s Guide).

templateoffset – The templateoffset attribute can be set on a template as x,y coordinates to alter the bounds of user drawn nodes when passed back into the ROES Server for DP2.
